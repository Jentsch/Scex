  8. case class Modifier[T](annotation: Annotation[T], value: T) extends Modifiers with Product with Serializable

    A tuple of an Annotation and a value, ready to be applied to a node.

    A tuple of an Annotation and a value, ready to be applied to a node. A single Modifier extends the seq type Modifiers because it allows all operations like Modifiers.

    val warning = TextColor > red

    Where TextColor is an Attribute (of type Color) and red a predefined color (both are part of scex.attributes.

    Inside a Builder it's now possiblie to do:

    text & warning | "Warning message"

    or even shorter:

    val warning = text & TextColor > red
warning"Warning message"
